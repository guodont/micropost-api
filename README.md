#sample micropost api

```
using rails 4.2.0
```

##Getting Started

###Install

```
    $git clone https://github.com/guodont/micropost-api.git

    $cd micropost-api/

    $bundle install

```

###Run

```
    $rails s
    $firefox [http:://localhost:3000](http:://localhost:3000)
```
###Docs

Restfull api docs
```
    $firefox [http://localhost:3000/docs/](http://localhost:3000/docs/)
```
